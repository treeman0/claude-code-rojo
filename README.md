# claude-code-rojo

A Claude Code plugin marketplace by treeman0.

## Plugins

### rojo-helper

Rojo integration for Claude Code. Adds slash commands for serving and building, a skill that teaches Claude Rojo project conventions, and an optional pre-edit warning hook.

## Install

In Claude Code:

```
/plugin marketplace add treeman0/claude-code-rojo
/plugin install rojo-helper@treeman0
```

## Requirements

- [Rojo](https://rojo.space) installed and on your `PATH`
- The Rojo Studio plugin installed in Roblox Studio

## Updating

Push new commits to this repo. Users get the latest version with:

```
/plugin marketplace update
```
